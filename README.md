# Quiz Application

A web-based Quiz Application that allows users to take multiple-choice quizzes, view their scores, and track progress. Built to be interactive, responsive, and easy to extend.

## Features
- Multiple-choice quizzes
- Score calculation and result display
- Responsive design for desktop and mobile
- Easy to add or modify quiz questions
